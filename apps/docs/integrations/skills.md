# Skills

**Skills** in Yanshi are configuration and instructions applied to agents — not executable
plugins. The Skills section is an honest aggregation of what actually exists.

::: tip Status — Available now <Badge type="tip" text="Available now" />
Skills list real data: each agent profile's instructions/personality (editable in
[Workshop → Agent Editor](/customization/agent-editor)) and installed Workshop packs with their
content types and enabled state.
:::

## What a skill is

- **Agent instructions** — the prompt/personality attached to an agent, injected into its
  execution context as advisory guidance.
- **Workshop packs** — reusable bundles (agent teams, office themes) you can enable/disable and
  export/import.

Each skill opens a detail modal showing its source, version, what it applies to, and an
instructions preview.

## What a skill is not

There is no executable plugin engine and no marketplace. Yanshi states plainly that skills are
instructions/config, so nothing implies code execution that does not happen.
