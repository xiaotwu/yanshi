# Agent Editor

The **Agent Editor** (Workshop → Agent Editor) edits your project's agent team.

## Per agent

- **Name** and **role/station** (Manager, Browser, Computer, File, Reviewer, Terminal).
- **Accent** color — the worker's individual mark in the Atelier.
- **Behavior mode** — professional / balanced / playful (affects idle liveliness, not task work).
- **Personality** and **Prompt** — advisory instructions injected into the agent's execution
  context, prompt-injection-separated. These also appear under [Skills](/integrations/skills).

## Persistence

Edits persist per project and survive restart. Exporting via [Workshop](/customization/workshop)
includes the agent team in the pack.

## Honest scope

Agent instructions are guidance applied to real agents — they shape prompts and context, they are
not executable plugins.
