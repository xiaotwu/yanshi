## Owner Revision

The owner clarified the desired abstraction: use the last supplied chibi image only for **Q-version
proportion, thick line weight, blank-cute expression, and simplified standing pose**, and design an
original dragon-horn girl mascot.

This does **not** authorize copying the referenced game character. The new direction allows dragon horns
as a Yanshi-original premise, but still forbids copied likeness: no reference horn silhouette, no long
black hair waterfall, no tactical outfit, no copied teal/crimson identity, no floating cross marks, no
name/lore/marks/assets from existing characters.

## Shared Chibi Language

- About two heads tall, with an oversized square-round head and tiny simplified body.
- Thick outline and simple interior strokes so the mascot survives rail-size thumbnails.
- Neutral face can use blank vertical eyes and a tiny mouth; happy face uses crescent eyes, blush, and a
  small open mouth.
- Standing pose stays simple: feet planted, short arms, minimal gesture shift between neutral/happy.
- Ancient workshop identity comes from apron, talisman, seal, brush, jade/brass ornaments, and small
  tool charms rather than tactical gear.
