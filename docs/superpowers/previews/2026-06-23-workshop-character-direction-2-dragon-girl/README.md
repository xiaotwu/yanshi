# Direction 2A Dragon-Horn Chibi Concepts

These are rendered concept previews only. They are not imported into the product and do not replace the
current tested `MascotRig`.

The previews use the requested Q-version abstraction: two-head proportions, thick outline, blank-cute
neutral face, happy face, and a simplified standing pose. They intentionally avoid copied character
identity: no reference horn silhouette, long hair waterfall, tactical outfit, copied palette, floating
marks, name, lore, or asset.
