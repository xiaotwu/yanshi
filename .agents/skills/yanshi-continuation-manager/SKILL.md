# Yanshi Continuation Manager

Always maintain:
- IMPLEMENTATION_PLAN.md
- IMPLEMENTATION_LOG.md
- CURRENT_STATUS.md
- NEXT_STEPS.md
- ACCEPTANCE_CHECKLIST.md

After every major phase:
- update current status
- record commands run
- record passing/failing checks
- record next action
- record blockers
- keep todo order clear

A new Codex session must be able to resume without guessing.
