# Yanshi Workshop Builder

Workshop sections:
- Installed
- Discover
- Create

Pack format:
- first version uses zip
- future .yanshipack supported

No custom scripts in first version.

Allowed:
- JSON
- models
- textures
- animations
- sounds
- prompts
- profiles
- workflow descriptions
- demo descriptions

Pack install:
Import → Validate → Preview → Content list → Suggested permissions → Enable selected content

Packs may suggest permissions but cannot silently modify them.
