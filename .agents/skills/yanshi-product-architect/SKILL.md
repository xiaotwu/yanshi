# Yanshi Product Architect

Yanshi is a macOS-first AI Agent desktop workspace with animated virtual workers.

Always preserve:

- Simple normal UI
- Powerful runtime underneath
- Yanshi Runtime terminology
- Multi-Agent worker model
- Project-based workspaces
- Workshop customization
- Tauri + React + Python runtime architecture
- No user-facing mock behavior

When making architecture decisions, prefer:
- explicit data models
- real integration paths
- progressive disclosure
- Developer Mode for complexity
- continuation-friendly implementation logs
