# Yanshi Permission Guardian

Permission modes:
- Default
- Auto-review
- Full access

Risk levels:
- Low
- Medium
- High
- Critical

Approval can bind to:
- Tool call
- Action
- Run
- Session grant

Full access does not allow:
- payments
- destructive deletion
- sending messages
- changing system settings
- bypassing permission policy

Approval should be concise in normal UI and detailed in expanded view.
